# Tekagon Admin Integration

Use this folder only if you want to show Gleank waitlist entries inside an existing Tekagon Admin React dashboard.

## Files

- `waitlistApi.js` — API functions for login/fetch/export
- `WaitlistPanel.jsx` — React component that displays the waitlist table
- `waitlist-panel.css` — simple styling

## Required environment variable in Tekagon frontend

```env
VITE_GLEANK_WAITLIST_API_URL=https://your-gleank-waitlist-backend.com/api
```

For local testing:

```env
VITE_GLEANK_WAITLIST_API_URL=http://localhost:5000/api
```

## Basic usage

```jsx
import WaitlistPanel from "./WaitlistPanel.jsx";
import "./waitlist-panel.css";

function AdminPage() {
  return <WaitlistPanel />;
}
```

The admin token must be generated by logging in through `/api/admin/login` using the same backend.
